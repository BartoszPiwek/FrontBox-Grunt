I. Padding/Margin

Make your own grid to make website flow.

.ml-42

.margin-20

.pb-34

m : margin

l : left

42 : how many px

margin : margin (left, top, right, bottom)

20 : how many px

p : padding

b : bottom

34 : how many px

Cheatsheet: How use padding/margin?

{property}{direction}-{count}px

{property} - margin, padding, m, p

{direction} - l, t, r, b

{count} - value

When use {property} margin or padding then don't need set direction becouse this two property apply in all direction value (left, top, right, bottom).

Only when use {property} m (margin) or p (padding) you must include {direction}.

Example:

.margin-50 .padding-10 .mb-14 .pr-100 .ml-2 .pt-6

Padding/Margin

Use class to add margin padding property for your elements. Replace {numer} with any number. Default numbers up to 100. You can change it in _untilizer.less

.padding-{number} - add padding with all property top, right, bottom, left

Example: .padding-20 (padding:20); .padding-14 (padding:14)

.pt-{number} .pb-{number} .pt-{number} .pr-{number} - add one direction padding: top, right, bottom, left

Example: .pt-20 (padding-top:20); .pl-54 (padding-left:54)

.margin-{number} - add margin with all property top, right, bottom, left

Example: .margin-20 (margin:20); .margin-14 (margin:14)

.mt-{number} .mb-{number} .mt-{number} .mr-{number} - add one direction margin: top, right, bottom, left

Example: .mt-20 (margin-top:20); .ml-54 (margin-left:54)
